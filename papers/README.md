# Papers

This directory contains 17 academic papers relevant to **Adversarial Prompt Propagation Dynamics in Multi-Agent LLM Collaborative Systems**.

## Paper Index

### Core Attack Papers (7)

1. **Prompt Infection: LLM-to-LLM Prompt Injection within Multi-Agent Systems**
   - ArXiv: [2410.07283](https://arxiv.org/abs/2410.07283)
   - File: `2410.07283_prompt_infection_llm_to_llm_multi_agent.pdf`
   - Key: Self-replicating prompt injection; logistic growth model; LLM Tagging defense

2. **Agent Smith: A Single Image Can Jailbreak One Million Multimodal LLM Agents**
   - ArXiv: [2402.08567](https://arxiv.org/abs/2402.08567)
   - File: `2402.08567_agent_smith_jailbreak_million_agents.pdf`
   - Key: SIR epidemiological model; exponential propagation; β ≤ 2γ containment criterion

3. **Tipping the Dominos: Topology-Aware Multi-Hop Attacks on LLM-Based Multi-Agent Systems**
   - ArXiv: [2512.04129](https://arxiv.org/abs/2512.04129)
   - File: `2512.04129_tipping_dominos_topology_multihop_attacks.pdf`
   - Key: TOMA attack; ACPM contamination model; HPES payload encapsulation; T-Guard defense

4. **Amplified Vulnerabilities: Structured Jailbreak on LLM-based Multi-Agent Debate**
   - ArXiv: [2504.16489](https://arxiv.org/abs/2504.16489)
   - File: `2504.16489_amplified_vulnerabilities_jailbreak_multiagent_debate.pdf`
   - Key: 185% amplification in MAD vs single-agent; narrative encapsulation attack

5. **MAD-Spear: Conformity-Driven Prompt Injection on Multi-Agent Debate Systems**
   - ArXiv: [2507.13038](https://arxiv.org/abs/2507.13038)
   - File: `2507.13038_mad_spear_conformity_prompt_injection_debate.pdf`
   - Key: Sybil-inspired attack; conformity pressure exploitation; MAD fault-tolerance

6. **OMNI-LEAK: Orchestrator Multi-Agent Network Induced Data Leakage**
   - ArXiv: [2602.13477](https://arxiv.org/abs/2602.13477)
   - File: `2602.13477_omni_leak_orchestrator_multiagent_data_leakage.pdf`
   - Key: Data exfiltration through orchestrator-based multi-agent networks

7. **The Wolf Within: Covert Injection of Malice into MLLM Societies**
   - ArXiv: [2402.14859](https://arxiv.org/abs/2402.14859)
   - File: `2402.14859_wolf_within_covert_injection_mllm_societies.pdf`
   - Key: Covert adversarial agent infiltration in MLLM societies

9. **NetSafe: Exploring the Topological Safety of Multi-agent Networks**
   - ArXiv: [2410.15686](https://arxiv.org/abs/2410.15686)
   - File: `2410.15686_netsafe_topological_safety_multiagent.pdf`

10. **G-Designer: Architecting Multi-agent Communication Topologies via GNN**
    - ArXiv: [2410.11782](https://arxiv.org/abs/2410.11782)
    - File: `2410.11782_g_designer_multiagent_communication_topologies.pdf`
    - Key: GNN-based topology optimization; performance-security tradeoffs
